# 電腦使用時間追蹤器

一個基於 Electron 的桌面計時應用程式，支援多用戶、活動分類和 Google Sheets 數據同步。

## 功能特點

### 🎯 核心功能
- **多用戶支援**：品瑜/品榕兩個用戶獨立計時
- **活動分類**：遊戲、YouTube、查資料、編寫文件、學習、其他
- **時間選擇**：5/10/15/20/30分鐘預設選項
- **即時統計**：顯示每日使用時間和次數

### 🎨 用戶界面
- **視覺設計**：漸變背景、圓形進度條、沙漏動畫
- **響應式設計**：自適應螢幕尺寸
- **成就系統**：專注時間達成里程碑提示
- **音效提示**：計時開始、暫停、完成音效

### 🏠 迷你模式
- **智能縮放**：計時時自動或手動進入迷你模式（160x60像素）
- **右上角顯示**：always-on-top，不干擾工作
- **智能攔截**：計時中點擊關閉(X)或最小化(-)自動進入迷你模式
- **一鍵返回**：點擊迷你窗口回到正常界面

### ☁️ 數據管理
- **Google Sheets 同步**：自動上傳計時記錄到雲端
- **本地備份**：離線時使用 localStorage 備份
- **自動載入**：啟動時載入今日統計數據
- **即時更新**：計時完成後立即同步
- **數據導出**：支援 CSV/JSON 格式匯出今日、本週或全部數據

### ⌨️ 全局快捷鍵 (v1.1.0)
- **Ctrl+Space**：快速開始/暫停計時
- **Ctrl+Shift+M**：切換迷你模式
- **Ctrl+Shift+R**：重置計時器
- **選單整合**：應用程式選單顯示快捷鍵

### 📊 統計圖表 (v1.1.0)
- **今日分佈圖**：用戶使用時間分佈餅圖
- **對比圖表**：用戶使用時間和次數比較
- **動態渲染**：HTML5 Canvas 即時繪製
- **主題適配**：圖表顏色隨主題自動調整

### 🌙 主題系統 (v1.1.0)
- **深色模式**：完整的深色主題支援
- **自動切換**：跟隨系統主題設定
- **設定記憶**：主題選擇本地化儲存
- **一鍵切換**：右上角主題切換按鈕

## 安裝與運行

### 環境需求
- Node.js 16+
- npm

### 安裝步驟
1. 安裝依賴
```bash
npm install
```

2. 運行應用程式
```bash
npm start
```

### Google Sheets 設置
1. 創建新的 Google Sheets
2. 打開 Apps Script (extensions.google.com/appsscript)
3. 複製 `google-apps-script-with-categories.js` 內容
4. 部署為 Web App，權限設為 "Anyone"
5. 複製 Web App URL 到 `src/index.html` 的 `SCRIPT_URL`

## 使用說明

### 基本操作
1. **選擇用戶**：點擊品瑜或品榕按鈕
2. **選擇活動**：點擊對應的活動類別
3. **設定時間**：選擇計時長度
4. **開始計時**：點擊開始按鈕，1秒後自動進入迷你模式
5. **迷你模式**：窗口縮小到右上角，顯示剩餘時間
6. **返回正常**：點擊迷你窗口回到正常界面

### 進階功能
- **音效控制**：右上角音效按鈕開關提示音
- **手動重載**：右上角重新載入按鈕同步雲端數據
- **智能攔截**：計時中點擊 X 或 - 會進入迷你模式而非關閉
- **數據導出**：📊 按鈕開啟匯出對話框，選擇格式和範圍
- **統計圖表**：📈 按鈕檢視今日使用分佈和用戶對比
- **主題切換**：🌙/☀️ 按鈕在深色/淺色模式間切換
- **全局快捷鍵**：任何時候使用 Ctrl+Space 快速操作

## 技術架構

### 主要技術
- **Electron**：跨平台桌面應用框架
- **HTML5/CSS3/JavaScript**：前端界面和邏輯
- **Google Apps Script**：後端數據處理
- **localStorage**：本地數據備份

### 關鍵特性
- **IPC 通信**：主進程與渲染進程通信
- **JSONP**：跨域 API 請求
- **高精度計時器**：防止系統休眠導致時間偏差
- **響應式設計**：適應不同螢幕尺寸
- **全局快捷鍵**：系統級快捷鍵註冊與管理
- **Canvas 繪圖**：動態圖表渲染與主題適配
- **CSS 變數**：動態主題切換系統
- **檔案下載**：Blob API 實現客戶端檔案導出

## 開發與部署

### 開發模式
```bash
NODE_ENV=development npm start
```
開發模式會自動開啟 DevTools

### 打包發布
```bash
npm run build
```
使用 electron-builder 打包為可執行文件

#### 支援平台
- **Windows**: 可攜式 .exe 檔案 (免安裝，雙擊即用)
- **Linux**: AppImage 檔案 (免安裝，雙擊即用)
- **macOS**: DMG 安裝包 (需在 macOS 環境建置)

#### 建置說明
- **Windows 環境**: 執行 `npm run build` 自動建立 Windows .exe
- **Linux 環境**: 執行 `npm run build` 自動建立 Linux AppImage
- **跨平台建置**: 需要在對應平台環境中執行

## 故障排除

### 常見問題
1. **CORS 錯誤**：確認使用 Electron 而非瀏覽器打開
2. **數據未同步**：檢查 Google Apps Script URL 是否正確
3. **窗口過小**：確認螢幕解析度支援最小窗口尺寸
4. **音效無聲**：檢查系統音量和應用程式音效設定

### 已修復問題 (v1.1.0)
5. **無法關閉應用程式** ✅：
   - 現在支援智能關閉邏輯：沒計時時按 X 直接關閉，計時中隱藏到托盤
   - 托盤右鍵「退出」和 Ctrl+Q 可強制關閉
6. **計時後無法縮放窗口** ✅：
   - 修復了最小尺寸設置衝突，現在可正常縮放 (最小 400x500)
7. **迷你模式大小異常** ✅：
   - 確保迷你模式固定為 160x60 像素，位於螢幕右上角

# 🕐 時間追蹤器 Universal v2.0.0

> **通用版本** - 支援Windows用戶自動登錄和多重存儲選擇

## 🎯 版本特色

### ✨ 全新功能
- **Windows用戶自動登錄**: 自動偵測並使用電腦用戶名稱
- **多重存儲選擇**: 本地、Google Sheets、雲端檔案
- **專業安裝程式**: NSIS標準安裝體驗
- **企業級界面**: 更專業的設計風格

### 🔄 與家庭版差異

| 功能 | 家庭版 v1.5.x | 通用版 v2.0.x |
|------|-------------|-------------|
| **用戶系統** | 固定A/B用戶 | Windows用戶自動登錄 |
| **資料存儲** | 指定Google Sheets | 多重選擇（本地/雲端） |
| **部署方式** | 免安裝版 | 標準安裝程式 |
| **目標用戶** | 家庭成員 | 一般用戶/企業 |

## 🚀 快速開始

### 開發環境
```bash
npm install
npm start
```

### 建置安裝程式
```bash
npm run build
```

## 📊 資料存儲方案

### 1. 本地存儲 (預設)
- 儲存位置: `%APPDATA%/時間追蹤器 Universal`
- 格式: JSON
- 優點: 隱私、速度、無需設定

### 2. 用戶Google Sheets
- 支援OAuth授權
- 完全用戶控制
- 支援多裝置同步

### 3. 本地檔案匯出
- 自動備份至用戶指定資料夾
- 支援JSON/CSV格式
- 可整合其他系統

## 🛠️ 技術架構

- **框架**: Electron 27.0+
- **用戶系統**: Node.js os.userInfo()
- **資料存儲**: 多重後端支援
- **建置工具**: electron-builder (NSIS)

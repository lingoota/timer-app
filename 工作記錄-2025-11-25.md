## 💡 重要學習筆記

### Git 權限問題
**問題**：電腦重灌後出現「dubious ownership」錯誤

**解決**：
```bash
git config --global --add safe.directory '[專案路徑]'
```

**原因**：Windows 使用者帳號改變，Git 認為目錄擁有者不同

### GitHub 遠端管理
**檢查遠端**：
```bash
git remote -v
```

**新增遠端**：
```bash
git remote add origin [URL]
```

**推送含標籤**：
```bash
git push origin main
git push origin [tag-name]
```

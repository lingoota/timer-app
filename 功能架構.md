# 時間追蹤器家庭版 — 功能架構圖

> **版本**: v2.0.0-beta.13 | **最後更新**: 2026-03-06

---

## 檔案結構

```
family-version/
├── main.js                  # Electron 主程序（視窗、IPC、更新、閒置偵測、迷你視窗）
├── package.json             # 專案配置與建置設定
├── src/
│   ├── index.html           # 主界面 HTML（所有 UI 元素）
│   ├── renderer.js          # 前端邏輯（計時、統計、用戶、Firebase）
│   ├── styles.css           # 全部樣式（含深色模式、背景遮罩、圖表卡片）
│   ├── preload.js           # IPC 橋接（expose window.api: invoke/on/send）
│   ├── firebase-helpers.js  # Firebase Firestore CRUD 操作
│   ├── background.js        # 背景圖片載入與遮罩控制
│   └── mini-window.html     # 迷你視窗（獨立無框 BrowserWindow）
└── assets/
    └── icon.png             # 應用程式圖示（Tray + BrowserWindow）
```

## 功能模組與關鍵函數

### 1. 計時系統
| 函數 | 檔案 | 說明 |
|------|------|------|
| `startTimer()` | renderer.js | 開始計時（含活動驗證） |
| `pauseTimer()` | renderer.js | 暫停計時 |
| `resetTimer()` | renderer.js | 重置計時器 |
| `updateTimer()` | renderer.js | 每秒更新（倒數、進度條、迷你模式同步） |
| `updateDisplay()` | renderer.js | 更新時間顯示與 SVG 圓形進度條 |
| `playSound(type)` | renderer.js | 音效播放（start/pause/reset/complete） |
| `stopAlarmSound()` | renderer.js | 停止警報音效 + 重置循環計數 |

計時完成音效循環：播 60s → 停 30s → 再播 60s → 第 2 次起強制視窗置頂

### 2. 用戶系統
| 函數 | 檔案 | 說明 |
|------|------|------|
| `initializeUserSelection()` | renderer.js | 身份綁定初始化（讀 `installedUser`） |
| `selectAndBindIdentity(identity)` | renderer.js | 首次選擇身份 → 永久記住 + 設定開機自啟 |
| `switchUser(user)` | renderer.js | 切換品瑜/品榕 |
| `initParentMode()` | renderer.js | 家長模式初始化（唯讀監控） |

用戶身份：`user1`(品瑜/藍) / `user2`(品榕/橙) / `parent`(家長)

### 3. 活動分類
7 種：🎮遊戲 📺YouTube 🔍查資料 📖看漫畫 📚學習 ✍️寫作 ⭐其他
DOM：`.activity-btn[data-activity]` → `state.selectedActivity`

### 4. Firebase 雲端同步
| 函數 | 檔案 | 說明 |
|------|------|------|
| `saveToFirebase()` | renderer.js | 計時完成 → 寫入 Firestore |
| `loadTodayStats()` | renderer.js | 載入今日統計 |
| `updateLiveTimerStatus()` | renderer.js | 每秒更新即時計時狀態（供家長監控） |
| Firebase CRUD | firebase-helpers.js | Firestore 底層操作封裝 |

路徑：`users/{pinyu|pinrong}/records`、`live-timer-status/{pinyu|pinrong}`

### 5. 統計圖表（beta.13 重做）
| 函數 | 檔案 | 說明 |
|------|------|------|
| `renderCharts()` | renderer.js | 統計主入口（更新摘要卡片 + 呼叫三個繪圖函數） |
| `renderDailyPieChart()` | renderer.js | 甜甜圈圓餅圖（Canvas 2D，中心顯示總分鐘） |
| `renderUserComparisonChart()` | renderer.js | 品瑜 vs 品榕圓角長條圖（分鐘 + 次數） |
| `renderHistoryTrend(days)` | renderer.js | 歷史趨勢折線圖（7/30天，雙色區域填充） |
| `getHistoryStats(days)` | firebase-helpers.js | 查詢多日歷史資料（Promise.all 並行） |
| `exportData(format)` | renderer.js | CSV/JSON 匯出 |

UI：摘要卡片（`#summary-pinyu/pinrong/total`）+ 圖表卡片（毛玻璃效果）+ 趨勢切換 tab（7/30天）

### 6. 迷你視窗（beta.12 獨立無框視窗）
| 函數 | 檔案 | 說明 |
|------|------|------|
| `createMiniWindow()` | main.js | 建立獨立無框 BrowserWindow（280x70） |
| `closeMiniWindow()` | main.js | 關閉迷你視窗（removeAllListeners 繞過攔截） |
| `enterMiniMode()` | renderer.js | 隱藏主視窗 → 發 `enter-mini-mode` IPC |
| `exitMiniMode()` | renderer.js | 發 `exit-mini-mode` IPC → 恢復主視窗 |

IPC：`enter-mini-mode`、`exit-mini-mode`、`mini-pause`、`mini-resume`、`mini-stop`、`mini-minimize`、`mini-close`
UI：mini-window.html（暫停/結束按鈕 + 最小化/關閉按鈕）、關閉隱藏到系統匣

### 7. 背景圖片
| 函數 | 檔案 | 說明 |
|------|------|------|
| `loadBackground()` | background.js | 從 localStorage 載入背景圖片 |
| `setBackground()` | background.js | 設定背景 + `#bg-overlay` 遮罩 rgba |
| `resetBackground()` | background.js | 右鍵重置背景 |
| `updateOverlayOpacity()` | background.js | 透明度滑桿控制 |

### 8. 閒置偵測提醒
| 函數 | 檔案 | 說明 |
|------|------|------|
| `setupActivityReminder()` | main.js | 啟動偵測（每 5s 檢查 `powerMonitor.getSystemIdleTime()`） |
| `stopActivityReminder()` | main.js | 停止偵測 |
| `setupReminderEvents()` | renderer.js | 綁定提醒對話框按鈕事件 |

邏輯：idle < 30s 且未計時 → 累計 `activeSeconds` → ≥60s 彈提醒（強制置頂）→ 5 分鐘冷卻

### 9. 開機自啟動
`selectAndBindIdentity()` → `window.api.send('set-auto-launch', isChild)`
main.js：`app.setLoginItemSettings({ openAtLogin: enabled })`

### 10. 手動更新
| 函數 | 檔案 | 說明 |
|------|------|------|
| `setupAutoUpdater()` | main.js | 初始化 electron-updater |
| `setupUpdateListeners()` | renderer.js | 綁定更新 UI 事件 |

### 11. UI 系統
| 元素 | 說明 |
|------|------|
| `.settings-wrapper` / `#settings-panel` | 齒輪面板（⚙️ 4x2 grid） |
| `.user-selection-overlay` | 首次身份選擇遮罩 |
| `#timer-reminder-overlay` | 閒置偵測提醒對話框 |
| `#bg-overlay` | 背景圖片遮罩層 |
| `prefers-color-scheme` | 深色模式自動跟隨 |

## IPC 通道清單

### 主程序 → 渲染進程
| 通道 | 用途 |
|------|------|
| `check-timer-status` | 詢問計時狀態（關閉/最小化前） |
| `global-shortcut` | 全局快捷鍵事件 |
| `show-timer-reminder` | 彈出閒置提醒（強制置頂） |
| `update-available/downloaded/not-available/error` | 更新狀態通知 |
| `download-progress` | 下載進度更新 |

### 渲染進程 → 主程序
| 通道 | 用途 |
|------|------|
| `fetch-google-script` | HTTP 請求代理（invoke） |
| `timer-status-response` | 回報計時狀態 |
| `update-mini-timer` | 更新迷你模式顯示（轉發到 miniWindow） |
| `timer-completed-show-window` | 計時完成還原視窗 |
| `enter-mini-mode` | 進入迷你模式（隱藏主視窗、建立迷你視窗） |
| `exit-mini-mode` | 退出迷你模式（關閉迷你視窗、顯示主視窗） |
| `set-auto-launch` | 設定開機自啟動 |
| `user-identity` | 回報用戶身份 |
| `timer-running-status` | 回報計時運行狀態 |
| `snooze-reminder` | 稍後提醒（5 分鐘冷卻） |
| `cancel-always-on-top` | 取消視窗置頂 |
| `force-window-top` | 強制視窗置頂（音效循環第 2 次） |
| `turn-off-screen` | 關閉螢幕 |
| `check-for-updates` / `download-update` / `quit-and-install` | 更新操作 |

### 迷你視窗 → 主程序
| 通道 | 用途 |
|------|------|
| `mini-pause` / `mini-resume` | 暫停/繼續計時 |
| `mini-stop` | 結束計時 |
| `mini-minimize` | 最小化迷你視窗 |
| `mini-close` | 隱藏迷你視窗到系統匣 |

### 主程序 → 迷你視窗
| 通道 | 用途 |
|------|------|
| `update-mini-display` | 更新時間顯示與用戶名稱 |
| `mini-pause-state` | 同步暫停狀態 |

## 資料儲存

| 位置 | 用途 |
|------|------|
| Firebase `users/{user}/records` | 計時記錄 |
| Firebase `live-timer-status/{user}` | 即時計時狀態 |
| `localStorage('installedUser')` | 綁定身份 |
| `localStorage('bgImage_user1/2')` | 背景圖片 Base64 |
| `localStorage('bgOpacity')` | 遮罩透明度 |
| `localStorage` 統計備份 | 離線備份計時記錄 |
| `window-state.json`（userData） | 主視窗尺寸/位置 + 迷你視窗位置 |

## 功能模組總覽

### 1. 計時系統 (`renderer.js`)
- 預設時間：5/10/15/20/30 分鐘按鈕
- 自定義時間：1-30 分鐘滑桿
- 開始/暫停/重置控制
- 圓形進度條即時顯示（SVG circle）
- 計時完成音效提示（Audio API）
- 高精度計時（`setInterval` 1 秒 + 系統時間校正）

### 2. 用戶系統 (`renderer.js`)
- 雙用戶：品瑜（user1，藍色）/ 品榕（user2，橙色）
- 身份綁定：首次啟動選擇身份 → `localStorage('installedUser')` 永久記住
- 家長模式：唯讀觀察兩位孩子的即時計時狀態
- 切換使用者：齒輪面板內提供切換按鈕

### 3. 活動分類 (`renderer.js`)
7 種活動類型，每次計時前選擇：
🎮 遊戲 | 📺 YouTube | 🔍 查資料 | 📖 看漫畫 | 📚 學習 | ✍️ 寫作 | ⭐ 其他

### 4. Firebase 雲端同步 (`firebase-helpers.js` + `renderer.js`)
- **寫入**：計時完成 → 存入 `users/{pinyu|pinrong}/records`
- **讀取**：啟動時載入今日/歷史統計
- **即時狀態**：計時中每秒更新 `live-timer-status/{user}`（供家長監控）
- **本地備份**：localStorage 雙保險，離線時不丟資料
- **安全規則**：僅允許 `pinyu`、`pinrong` 路徑讀寫

### 5. 統計圖表 (`renderer.js`)
- 圓餅圖：今日各活動時間佔比（Canvas 2D）
- 長條圖：過去 7 天每日使用時間
- 24 小時時間線：當日各時段使用分佈
- CSV/JSON 匯出

### 6. 迷你模式 (`main.js` + `renderer.js`)
- 正常視窗 ↔ 迷你窗口（160x60）切換
- always-on-top 置頂顯示
- 顯示剩餘時間和用戶名稱
- 計時中攔截關閉/最小化（防止意外中斷）

### 7. 背景圖片 (`background.js` + `styles.css`)
- 品瑜/品榕各自專屬背景（PNG/JPG，最大 5MB）
- `#bg-overlay` 整頁遮罩層（rgba 半透明）
- 毛玻璃效果（`backdrop-filter: blur`）
- 深色/淺色模式自動調整遮罩不透明度
- 透明度滑桿（齒輪面板內）
- 右鍵背景圖片 → 重置

### 8. 閒置偵測提醒 (`main.js`)
- `powerMonitor.getSystemIdleTime()` 每 5 秒檢查
- 電腦使用中（idle < 30s）且未計時 → 累計 `activeSeconds`
- 超過 60 秒彈出提醒對話框
- 「開始計時」/ 「稍後提醒（5 分鐘冷卻）」
- 僅小孩身份啟用，家長模式不啟用

### 9. 開機自啟動 (`main.js`)
- 小孩身份選擇後 → `app.setLoginItemSettings({ openAtLogin: true })`
- 家長模式不自動啟動

### 10. 手動更新 (`main.js` + `renderer.js`)
- `electron-updater` + GitHub Releases
- 齒輪面板內「檢查更新」按鈕
- 發現新版本 → 下載進度條 → 重啟安裝提示
- 必須上傳 `.exe` + `.exe.blockmap` + `latest.yml`

### 11. UI 系統 (`styles.css` + `index.html`)
- 齒輪面板（⚙️）：8 個功能按鈕收折為 4x2 grid
- 深色模式：`prefers-color-scheme` 系統主題跟隨
- 全域文字陰影：提升背景圖片上的文字可讀性
- 全局快捷鍵：`Ctrl+Space`（開始/暫停）等

## IPC 通道清單

### 主程序 → 渲染進程（`main.js` → `renderer.js`）
| 通道 | 用途 |
|------|------|
| `check-timer-status` | 詢問計時狀態（關閉/最小化前） |
| `timer-minimized` | 通知已切換迷你模式 |
| `timer-restored` | 通知已還原正常模式 |
| `global-shortcut` | 全局快捷鍵事件 |
| `show-timer-reminder` | 彈出閒置提醒對話框 |
| `update-available` | 發現新版本 |
| `download-progress` | 下載進度更新 |
| `update-downloaded` | 下載完成 |
| `update-not-available` | 已是最新版 |
| `update-error` | 更新錯誤 |

### 渲染進程 → 主程序（`renderer.js` → `main.js`）
| 通道 | 用途 |
|------|------|
| `fetch-google-script` | HTTP 請求代理（invoke） |
| `timer-status-response` | 回報計時狀態 |
| `update-mini-timer` | 更新迷你模式顯示 |
| `timer-completed-show-window` | 計時完成還原視窗 |
| `restore-timer` | 還原正常模式 |
| `sync-mini-mode-state` | 同步迷你模式狀態 |
| `set-auto-launch` | 設定開機自啟動 |
| `user-identity` | 回報用戶身份 |
| `timer-running-status` | 回報計時運行狀態 |
| `snooze-reminder` | 稍後提醒（5 分鐘冷卻） |
| `check-for-updates` | 手動檢查更新 |
| `download-update` | 開始下載更新 |
| `quit-and-install` | 退出並安裝更新 |

## 資料儲存

| 位置 | 用途 | 格式 |
|------|------|------|
| Firebase Firestore | 計時記錄、即時狀態 | Document/Collection |
| localStorage `installedUser` | 綁定身份 | string |
| localStorage `rememberedUser` | 上次使用的用戶 | string |
| localStorage `bgImage_user1/2` | 背景圖片 Base64 | string |
| localStorage `bgOpacity` | 遮罩透明度 | number |
| localStorage 統計備份 | 離線備份計時記錄 | JSON |

## 🎯 核心功能清單

### ✅ 已完成功能 (v1.5.0)

#### 🕐 計時系統
- [x] **基礎計時**: 開始/暫停/重置功能
- [x] **預設時間**: 5/10/15/20/30分鐘選項
- [x] **自定義時間**: 1-30分鐘任意設定 ⭐️ v1.5.0新增
- [x] **進度顯示**: 圓形進度條即時顯示
- [x] **高精度計時**: 防止系統休眠影響

#### 👥 用戶管理
- [x] **雙用戶系統**: A用戶/B用戶獨立切換
- [x] **獨立統計**: 各用戶使用時間和次數統計
- [x] **視覺區分**: 藍色/橙色用戶識別

#### 📚 活動分類
- [x] **7大類活動**: 遊戲、YouTube、查資料、看漫畫、學習、✍️寫作、其他 ⭐️ v1.5.1新增寫作
- [x] **數據記錄**: Google Sheets雲端同步
- [x] **統計圖表**: 活動分佈餅圖、用戶對比長條圖

#### 🔧 界面功能
- [x] **迷你模式**: 160x60像素小窗口，always-on-top
- [x] **智能窗口**: 計時中攔截關閉/最小化操作
- [x] **深色模式**: 系統主題跟隨
- [x] **全局快捷鍵**: Ctrl+Space開始/暫停等

#### 📊 數據管理
- [x] **雲端同步**: Google Sheets自動同步
- [x] **本地備份**: localStorage備份機制
- [x] **數據匯出**: CSV/JSON格式匯出
- [x] **錯誤處理**: 完善的重試機制

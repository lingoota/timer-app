## 🎯 核心目標（長期實現）

讓家中兩個女兒（品瑜/品榕）自我管理電腦使用時間：計時、分類記錄活動、用完強制休息、雲端同步讓家長遠端觀察。**新手友善、簡單優先**，小孩能獨立操作。

---

## 🧭 核心原則（本專案）

- **簡單優先、新手友善**：UI 與互動以小孩能獨立使用為準，不堆功能
- **計時準確凌駕一切**：計時是核心，寧可三重保險（main 進程 tick + powerSaveBlocker + backgroundThrottling）也不能在背景被節流卡住
- **休息要有「彈性對應」**：休息時間跟實際使用掛鉤（用多久休多久），不是固定值；比例/上下限保留調整空間
- **手動更新優於自動更新**：更新可控、不打斷小孩使用
- **資料雙保險**：Firebase 雲端 + localStorage 本地備份並存

## ✅ 現有功能

- **計時系統**：開始/暫停/重置、預設 5/10/15/20/30 分、自定義 1-30 分、圓形進度條、高精度計時（main 進程 tick 不被 Chromium 節流）、完成音效
- **休息冷卻期**：用多久休多久（1:1，依實際使用時間，不設上下限）
- **雙用戶系統**：品瑜/品榕獨立切換與統計、藍/橙視覺區分、身份綁定永久記住、開機自啟動
- **8 種活動分類**：🎮遊戲 📺YouTube 🔍查資料 📖看漫畫 📚學習 ✍️寫作 🎨繪畫 ⭐其他
- **Firebase 雲端同步**：即時同步、品瑜/品榕資料隔離、離線支援、localStorage 本地備份、正式安全規則
- **家長觀察模式**：唯讀查看兩童今日統計、即時監控正在計時的孩子（倒數 + 進度 %，每秒更新）
- **更新機制**：檢查更新按鈕 + 啟動自動檢查（延遲 5 秒）+ 下載進度 + 重啟安裝（electron-updater + GitHub Releases）
- **個人化背景**：各用戶專屬背景圖、毛玻璃效果、深淺色自適配遮罩
- **迷你視窗**：獨立無框小視窗（280x70），可拖曳、記位置、暫停/結束、位置超界自動重置
- **其他**：智能窗口管理（計時中攔截關閉/最小化）、深淺色模式（白/黑字）、全局快捷鍵、閒置偵測提醒、關閉螢幕、統計圖表（圓餅/長條/趨勢）、CSV/JSON 匯出、單一實例鎖定

## 📝 重要技術決策

| 決策項目 | 選擇 | 原因 |
|---------|------|------|
| 雲端資料庫 | Firebase Firestore（取代 Google Sheets） | 即時同步、免費、離線支援、資料隔離完整 |
| 更新方式 | 手動更新 + 啟動自動檢查（非全自動） | 自動更新不穩定、會打斷小孩使用 |
| 安裝形式 | NSIS 安裝版（取代免安裝版） | 免安裝版無法用 electron-updater |
| 計時 tick | 搬到 main 進程（Node.js setInterval） | renderer 的 setInterval 在 Chromium 全螢幕背景被嚴重節流，計時器會卡住 |
| 休息時間 | 用多久休多久（1:1，實際使用時間） | 固定 5 分鐘不合理；護眼概念，使用越久越該休息 |

## ⚠️ 重要注意事項 / 鐵律

- **發版必傳三件**：`.exe` + `.exe.blockmap` + `latest.yml`，缺 latest.yml 自動更新偵測不到
- **不可標 Pre-release**：electron-updater 預設只看 "Latest" Release，Pre-release 被忽略
- **Repo 必須 Public**：Private repo electron-updater 讀 Release 會 404
- **Electron 無法在 Claude Code 沙盒啟動**：`app` 為 undefined，功能測試必須使用者手動 `npm start`
- **Firebase API Key 公開無風險**：安全靠 Security Rules（已更新為正式規則，2026-02-26）

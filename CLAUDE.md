# 🕐 時間追蹤器專案 - Claude Code 專案說明檔

## 📋 專案概覽

**專案名稱**: 時間追蹤器 (Time Calculator)  
**當前版本**: v1.5.0  
**技術架構**: Electron + HTML/CSS/JavaScript  
**開發環境**: Windows 11 + WSL + VS Code  
**目標用戶**: 需要追蹤個人時間使用的用戶

## 🎯 核心功能

### ✅ 已完成功能 (v1.5.0)
- **靈活計時系統**: 5/10/15/20/30分鐘預設 + 1-30分鐘自定義
- **多用戶支援**: A用戶/B用戶切換
- **活動分類**: 遊戲、YouTube、查資料、看漫畫、學習、其他
- **統計圖表**: 活動類別統計圖表系統
- **數據匯出**: 支援匯出統計數據
- **迷你模式**: 精簡界面模式

### 🔧 技術特色
- **Electron桌面應用**: 跨平台桌面應用程式框架
- **免安裝版本**: Windows x64 免安裝執行檔
- **現代化界面**: 毛玻璃效果、響應式設計
- **數據持久化**: 本地儲存用戶數據

## 🛠️ 開發環境設定

### WSL環境配置
```bash
# 檢查Node.js版本
node --version

# 安裝依賴
npm install

# 開發模式運行
npm start

# 建置應用程式 (需在Windows環境)
npm run build-portable
```

### 重要注意事項
⚠️ **WSL限制**: Electron建置需在Windows環境執行  
✅ **開發可行**: WSL可正常進行程式開發和測試  
🔄 **建置流程**: 程式碼完成後需切換到Windows環境建置

## 🔧 故障排除

### 常見問題
1. **Electron無法啟動**: 檢查Node.js版本兼容性
2. **建置失敗**: 確保在Windows環境執行建置指令
3. **依賴錯誤**: 刪除node_modules後重新安裝

### WSL特殊注意事項
- **GUI應用**: Electron在WSL中可開發但建置需Windows
- **檔案路徑**: 使用絕對路徑避免WSL/Windows路徑混淆
- **權限問題**: 確保檔案有適當的讀寫權限

# 電腦使用時間追蹤器 - 現有功能概覽

本文件記錄了當前「電腦使用時間追蹤器」應用程式所實現的主要功能，以供未來開發參考。

## 核心功能

1.  **使用者切換**：
    *   應用程式支援在兩個預設使用者（「品瑜」和「品榕」）之間進行切換。
    *   不同使用者的計時數據會獨立記錄和統計。

2.  **計時功能**：
    *   使用者可以選擇預設的時間長度（5、10、15、20、30 分鐘）或自定義時間（1-30分鐘）作為計時目標。
    *   可以選擇不同的活動類別來標記計時內容（例如：遊戲、YouTube、查資料、看漫畫、學習、其他）。
    *   計時器啟動後，會顯示一個圓形進度條，視覺化地展示剩餘時間。
    *   提供「開始」、「暫停」和「重置」按鈕來控制計時器的狀態。

3.  **數據統計與顯示**：
    *   主介面會即時顯示當前使用者和總計的今日使用時間及會話次數。
    *   提供統計圖表功能，可視覺化展示：
        *   **當前用戶活動分佈**：以餅圖形式顯示當前用戶各活動類別的時間分佈和百分比。
        *   **用戶使用對比**：以柱狀圖形式比較兩個用戶的總時間和會話次數。

## 特色功能

1.  **迷你模式 (Mini Mode)**：
    *   當計時器運行時，應用程式可以切換到一個小型的、置頂的迷你視窗。
    *   迷你視窗僅顯示剩餘時間和當前使用者，方便使用者在進行其他工作時監控計時。
    *   點擊迷你視窗可恢復到正常模式。

2.  **數據同步與匯出**：
    *   計時數據會自動同步到 Google Sheets，實現雲端備份和跨設備數據一致性。
    *   支援將數據匯出為 `CSV` 或 `JSON` 格式。
    *   匯出範圍可選擇「今日」、「本週」或「所有」數據。

3.  **使用者體驗優化**：
    *   **音效提示**：在計時開始、暫停和完成時提供音效反饋。
    *   **視覺動畫**：包含沙漏動畫和計時完成動畫，增加互動性和趣味性。
    *   **主題切換**：支援淺色和深色兩種應用程式主題，使用者可手動切換或跟隨系統主題。
    *   **智慧視窗行為**：當計時器運行時，關閉主視窗會將應用程式隱藏到系統托盤，而不是直接退出，以避免計時中斷。

4.  **全域快捷鍵**：
    *   `Ctrl + Space`：快速開始/暫停計時。
    *   `Ctrl + Shift + M`：切換迷你模式。
    *   `Ctrl + Shift + R`：重置計時器。

## 技術棧

*   **Electron**：用於構建跨平台桌面應用程式。
*   **HTML/CSS/JavaScript**：前端界面和邏輯。
*   **Tailwind CSS**：用於快速構建響應式 UI (通過 CDN 引入)。
*   **HTML5 Canvas**：用於數據視覺化（圖表渲染）。
*   **Google Apps Script**：作為後端服務，處理與 Google Sheets 的數據交互。
*   **Axios**：處理HTTP請求和Google Apps Script通信。

### 2025-06-28: 重要問題修復與改進

#### 🐛 Google Sheets 連動修復
- **問題**: 應用程式啟動時無法正常載入今日使用記錄
- **原因**: `renderer.js:339` 存在JavaScript語法錯誤 - CSS選擇器缺少結尾括號
- **修復**: 
  ```javascript
  // 修復前
  const range = document.querySelector('input[name="export-range']:checked').value;
  // 修復後  
  const range = document.querySelector('input[name="export-range"]:checked').value;
  ```

## 系統穩定性

### 錯誤處理機制
- **三層錯誤處理**: 本地備份 → 網路請求失敗 → 解析錯誤
- **智能降級**: 網路問題時自動使用本地數據
- **用戶友好提示**: 清楚的狀態訊息和錯誤說明

### 數據一致性
- **雲端優先**: 啟動時優先載入Google Sheets數據
- **本地備份**: localStorage作為備援機制
- **同步狀態**: 即時顯示同步狀態和錯誤提示

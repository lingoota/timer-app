# /完成 - 標記當前工作為完成

## 指令說明
標記當前工作為完成並執行：
1. 更新 todo.md
2. 更新 專案概況.md 的進度
3. 記錄學到的重點
4. 準備下一個任務

## 執行內容

請執行以下操作：

1. **確認完成的工作**
   - 詢問用戶剛完成了什麼工作
   - 確認是否需要測試驗證
   - 檢查相關檔案是否已更新

2. **更新文檔**
   - 在 TODO.md 中將對應項目標記為完成 ✅
   - 更新 專案概況.md 的進度資訊
   - 記錄完成時間和重要細節

3. **學習記錄**
   - 詢問這次工作學到了什麼
   - 是否遇到什麼特別的問題
   - 有什麼經驗可以記錄

4. **準備下一步**
   - 檢查 TODO 中的下一個優先任務
   - 評估是否需要立即處理
   - 提供開始下一個任務的建議

## 使用時機
- 完成一個功能開發後
- 修復一個問題後
- 完成一次重要更新後

## 輸出格式
```
✅ 工作完成確認

🎯 完成項目: [工作項目名稱]
📅 完成時間: [時間戳記]
💡 重要學習: [學到的重點]

📋 文檔更新:
- ✅ TODO.md 已更新
- ✅ 專案概況.md 已更新
- ✅ 進度記錄已保存

🚀 下一步建議:
- [下一個建議任務]

準備好開始下一個項目了嗎？
```

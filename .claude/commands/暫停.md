# /暫停 - 暫停當前工作

## 指令說明
暫停當前工作並執行：
1. 記錄目前進度
2. 更新 todo.md 的暫停狀態
3. 更新 專案概況.md 的當前狀態
4. 記錄下次繼續的提醒

## 執行內容

請執行以下操作：

1. **記錄當前進度**
   - 詢問用戶目前工作到哪裡
   - 記錄已完成的部分
   - 記錄遇到的問題或阻礙

2. **更新狀態**
   - 在 TODO.md 中標記項目為暫停 ⏸️
   - 記錄暫停原因和當前進度
   - 更新 專案概況.md 的最後更新時間

3. **保存工作現場**
   - 記錄正在編輯的檔案
   - 記錄當前的思路和想法
   - 記錄下次需要從哪裡開始

4. **下次繼續提醒**
   - 記錄需要回顧的檔案
   - 記錄需要測試的內容
   - 記錄任何特殊注意事項

## 使用時機
- 需要暫時停止工作時
- 遇到阻礙需要思考時
- 工作時間結束時

## 輸出格式
```
⏸️ 工作暫停記錄

🎯 暫停項目: [工作項目名稱]
📅 暫停時間: [時間戳記]
📊 完成進度: [進度百分比]

🔄 當前狀態:
- 已完成: [已完成的部分]
- 進行中: [正在處理的部分]
- 遇到問題: [遇到的困難]

📝 下次繼續提醒:
- 從這裡開始: [具體位置]
- 需要回顧: [相關檔案]
- 注意事項: [特殊提醒]

💡 工作思路:
- [當前想法和計劃]

使用 /繼續 來恢復工作
```

# /狀態 - 快速檢視專案狀態

## 指令說明
快速檢視專案整體狀態，包括：
- 專案整體進度
- Git 狀態（未提交的變更）
- 最近的工作重點
- 待解決的問題

## 執行內容

請執行以下操作：

1. **Git 狀態檢查**
   ```bash
   git status --porcelain
   git log --oneline -5
   ```

2. **專案進度概覽**
   - 讀取 package.json 確認當前版本
   - 檢查 TODO.md 中進行中和待辦項目數量
   - 查看最近完成的里程碑

3. **檔案狀態檢查**
   - 檢查是否有未提交的重要變更
   - 確認建置狀態
   - 檢查依賴是否正常

## 輸出格式
```
📊 專案狀態快報

🎯 專案資訊:
- 版本: v1.5.0
- 狀態: [當前階段]
- 完成度: XX%

🔄 Git 狀態:
- 分支: main
- 未提交變更: X 個檔案
- 最近提交: [最近的commit]

📋 工作進度:
- 已完成: XX 項
- 進行中: X 項  
- 待辦: XX 項

⚡ 快速動作:
- 使用 /繼續 開始工作
- 使用 /推送 提交變更
```

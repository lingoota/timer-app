# /繼續 - 查看現況並繼續工作

## 指令說明
自動執行以下步驟：
1. 讀取 `todo.md` 了解當前進度
2. 檢查最近的 commit 和檔案變更
3. 列出專案現況摘要
4. 詢問要從哪個項目開始繼續

## 執行內容

請執行以下操作：

1. **讀取當前狀態**
   - 查看 TODO.md 了解待辦事項
   - 檢查 Git 狀態和最近變更
   - 讀取專案概況.md 了解整體進度

2. **生成現況摘要**
   - ✅ **已完成的項目**: 列出最近完成的工作
   - 🚧 **進行中的工作**: 顯示當前正在處理的任務
   - 📝 **下一步要做的事情**: 從TODO中選出優先事項
   - 💡 **需要注意的重點**: 特殊注意事項或潛在問題

3. **準備繼續工作**
   - 詢問用戶想要處理哪個具體任務
   - 確保開發環境準備就緒
   - 提供相關檔案路徑和指令

## 使用時機
- 重新開始工作時
- 想了解專案當前狀況時
- 需要選擇下一個工作項目時

## 輸出格式
```
🔄 專案現況摘要

✅ 已完成項目:
- [完成的工作項目]

🚧 進行中工作:
- [當前進行的任務]

📝 下一步計劃:
1. [優先任務1]
2. [優先任務2]
3. [優先任務3]

💡 重要提醒:
- [需要注意的事項]

請選擇要繼續的工作項目...
```

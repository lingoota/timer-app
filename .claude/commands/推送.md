# /推送 - 一鍵推送到GitHub

## 指令說明
一鍵完成所有更新和推送：
1. 檢查 Git 狀態和未提交的變更
2. 自動執行 `git add .`
3. 要求輸入繁體中文 commit message
4. 執行 `git commit` 和 `git push`
5. 自動更新專案文檔記錄推送時間
6. 更新 todo.md 標記版本里程碑

## 執行內容

請執行以下操作：

1. **檢查Git狀態**
   ```bash
   git status
   git diff --stat
   ```
   - 顯示所有變更的檔案
   - 確認沒有需要特殊處理的檔案

2. **準備提交**
   ```bash
   git add .
   ```
   - 詢問用戶提交訊息（必須是繁體中文）
   - 檢查提交訊息是否清楚描述變更內容

3. **執行提交和推送**
   ```bash
   git commit -m "$(cat <<'EOF'
   [用戶輸入的提交訊息]
   
   🤖 Generated with [Claude Code](https://claude.ai/code)
   
   Co-Authored-By: Claude <noreply@anthropic.com>
   EOF
   )"
   git push origin main
   ```

4. **更新專案文檔**
   - 更新 專案概況.md 記錄推送時間和內容
   - 如果是版本發布，更新 TODO.md 標記里程碑
   - 記錄這次推送的重要變更

5. **確認推送成功**
   ```bash
   git status
   git log --oneline -1
   ```

## 使用時機
- 完成一個功能後
- 修復重要問題後
- 需要備份工作進度時
- 準備版本發布時

## 輸出格式
```
🚀 Git 推送流程

📋 變更檔案:
- [列出修改的檔案]

💬 提交訊息: 
- [用戶輸入的訊息]

✅ 推送結果:
- 提交成功: [commit hash]
- 推送成功: origin/main
- 推送時間: [時間戳記]

📊 文檔更新:
- ✅ 專案概況.md 已更新
- ✅ 推送記錄已保存

🎯 下一步:
- 使用 /狀態 檢查專案狀態
- 使用 /繼續 開始新工作
```

## 重要注意事項
- 確保提交訊息使用繁體中文
- 每次推送都會自動加上 Claude Code 標記
- 推送前會檢查是否有衝突
- 失敗時會提供詳細錯誤訊息
